# Sneaker Shop Website

## Overview
This project is a database-driven website developed as part of a university assignment. The website allows users to browse sneakers and enables registered members to add new products through a secure system.

## Features
- User registration and login system
- Secure authentication using password hashing
- Member-only access to add new sneakers
- Dynamic product display from database
- Search functionality using keywords
- Navigation system across multiple pages

## Technologies Used
- PHP (server-side scripting)
- MySQL (database)
- HTML5 & CSS3 (frontend)
- Git & GitHub (version control)

## Security Features
- Prepared statements to prevent SQL injection
- Password hashing using `password_hash()`
- Session-based authentication
- Restricted access to member-only pages


## How to Run the Project
1. Place project folder in:
   - `C:\wamp64\www\`
2. Start WAMP server
3. Open browser:
   - `http://localhost/sneaker-shop`

## Database Setup
- Create database: `sneaker_shop`
- Import tables:
  - `users`
  - `sneakers`
